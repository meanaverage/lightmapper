LightMapper API Documentation

This is the REST API documentation for LightMapper - an advanced Home Assistant lighting controller with interactive floorplan editor.

üîê Authentication

To use the external API endpoints, you need to:

  1. Enable external API access in the add-on configuration: enable_external_api: true
  2. Set an API key in the configuration: api_key: "your-secret-key"
  3. Include the API key in your requests:
    • Header: X-API-Key: your-secret-key
    • Query: ?api_key=your-secret-key
